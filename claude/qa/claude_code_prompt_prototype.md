# Claude Code プロンプト：不動産Q&A AIプラットフォーム インタラクティブプロトタイプ

## 目的

ICTスタートアップリーグ（総務省助成金）応募用の3分プレゼンテーション動画で画面録画するための、不動産Q&A AIプラットフォームのインタラクティブプロトタイプをReact（Next.js）で作成してください。

**重要：これは実際に動作するプロダクトではなく、デモ動画用のプロトタイプです。** バックエンドは不要で、すべてフロントエンドのモック（ハードコードされた回答のタイピングアニメーション等）で構成してください。

---

## プロジェクト概要

**サービス名：** AIマッチング専門家QA（仮称）
**コンセプト：** 不動産ドメイン特化RAGと専門家のハイブリッドモデルで、消費者の不動産に関する疑問・トラブルに低コスト（月額500円〜）で回答するプラットフォーム

**ビジネスモデル（パターンC）：**
- 無料層：過去のQ&A閲覧・検索のみ
- サブスク（月額500円）：AI即時回答の利用、質問投稿
- スポット課金（1質問500〜5,000円）：専門家確認付き個別回答

**技術的核心：**
- ユーザーの質問をNLPで分類（簡易/複雑）
- 簡易質問 → ドメイン特化RAGが即時回答（法規条文・判例を根拠として引用）
- 複雑質問 → RAGが下書きを生成し、専門家が確認・編集して回答

### 画面2：AI即時回答画面（簡易質問パス）

**この画面が動画のハイライト。RAGの価値を最も伝える画面。**

**レイアウト：**
- 左上に質問の難易度バッジ：「📊 AI判定：一般的な質問（FAQ型）」（緑バッジ）
- ユーザーの質問（カード形式）
- AI回答エリア（チャット形式ではなく、記事形式のリッチレイアウト）：

**AI回答の内容（ハードコード）：**

```
## 回答

敷金の返還と原状回復費用については、以下のルールが適用されます。

### 1. 敷金は原則として全額返還されます

賃貸借契約が終了した場合、賃貸人（大家）は敷金から賃借人の債務（未払い賃料等）
を差し引いた残額を返還する義務があります。

📎 根拠：民法第622条の2第1項
「賃貸人は、敷金を受け取っている場合において、（中略）その受け取った敷金の額
から賃貸借に基づいて生じた賃借人の賃貸人に対する金銭の給付を目的とする債務の
額を控除した残額を返還しなければならない。」

### 2. 通常損耗（経年劣化）は貸主負担です

普通に生活していて生じる損耗（壁紙の日焼け、畳の摩耗、家具設置によるカーペット
のへこみ等）は、原状回復の対象外です。

📎 根拠：国土交通省「原状回復をめぐるトラブルとガイドライン」（再改訂版）
「賃借人の居住、使用により発生した建物価値の減少のうち、賃借人の故意・過失、
善管注意義務違反、その他通常の使用を超えるような使用による損耗・毀損を復旧する
こと」が原状回復の定義。

### 3. 特約がある場合の注意点

契約書に「ハウスクリーニング費用は借主負担」等の特約がある場合、一定の条件を
満たせば有効とされます。ただし、消費者契約法第10条により、消費者の利益を
一方的に害する条項は無効となる場合があります。

📎 参考判例：最高裁平成17年12月16日判決
通常損耗補修特約が成立するためには、賃借人が補修費用を負担することになる通常
損耗の範囲が賃貸借契約書等に具体的に明記されている必要がある。

---

⚠️ この回答はAIが法規・判例データベースから自動生成したものです。
個別の契約内容や地域の慣行により異なる場合があります。
より詳しい相談は「専門家に質問する」をご利用ください。
```

**右サイドバー：**
- 「この回答は役に立ちましたか？」👍 👎
- 「関連する質問」（3件リスト）
- 「より詳しく専門家に相談する →」ボタン（目立つCTA）
- 「引用された法規・判例」セクション（条文番号のリンク風表示）

**デモ用インタラクション：**
- 回答がセクションごとにフェードイン（タイピングアニメーションまたはストリーミング風の逐次表示）
- 📎マークの根拠部分がハイライトされて目立つ
- 表示完了後、「専門家に質問する」ボタンをクリックして画面3へ

- 追加情報フォーム：

### 画面4：専門家回答画面（ユーザー視点）

**レイアウト：**
- 回答ステータスタイムライン：
  「質問送信 ✓」→「AI下書き生成 ✓」→「専門家確認・編集 ✓」→「回答完了 ✓」
- 専門家回答（記事形式）：

```
## 専門家回答

回答者：田中 太郎（宅地建物取引士）
回答日：2025年7月15日

### 結論

6年間居住後のクリーニング特約3万円の請求は、争う余地があります。

### 詳細

お客様のケースでは以下の点を考慮する必要があります：

1. **経年劣化の考慮**
   6年居住の場合、壁紙・カーペット等の残存価値は大幅に低下しています。
   ガイドラインの耐用年数表に基づくと、6年でほぼ残存価値1円となります。

2. **クリーニング特約の有効性**
   最高裁判例（H17.12.16）に照らすと、特約自体は具体的な金額が
   明記されている場合、有効とされる可能性が高いです。
   ただし、3万円という金額の妥当性は別途検討が必要です。

3. **東京都の条例**
   東京都「賃貸住宅紛争防止条例」（東京ルール）により、契約時に
   退去時の費用負担について書面で説明を受けている必要があります。

### 推奨アクション

まず契約書の特約条項を確認し、「具体的な範囲と金額」が明記されているか
確認してください。記載が曖昧な場合は、特約の有効性を主張して減額交渉が
可能です。交渉が難航する場合は、消費生活センター（188番）への相談を
お勧めします。
```

- 回答下部に「✅ 専門家が確認済み」のバッジ

**デモ用インタラクション：**
- タイムラインが順番にアニメーションで完了していく
- 回答がフェードインで表示

## 技術要件

### スタック
- **Next.js 14+**（App Router）
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion**（アニメーション）

### アニメーション要件（デモ動画用に重要）
1. **タイピングアニメーション：** 画面1の検索バーへの質問入力、画面2のAI回答のストリーミング風表示
2. **フェードイン：** 各セクションの順次表示（stagger effect）
3. **スライドイン：** サイドバー要素、カード要素
4. **プログレスバー：** 画面4のステータスタイムライン
5. **ハイライト：** 根拠条文の引用部分が光る（パルスアニメーション）

### 画面遷移
- 各画面間はボタンクリックで遷移（Next.jsのルーティング）
- 画面録画時にスムーズに操作できるよう、適切なタイミングで自動遷移オプションも用意（URLパラメータ `?auto=true` で自動デモモード）
- 自動デモモード：画面1（8秒：タイピング）→ 画面2（15秒：回答表示）→ 画面3（5秒）→ 画面4（10秒：タイムライン）→ 画面5（8秒）→ 画面6（5秒）

### レスポンシブ
- デモ動画用なので **デスクトップ表示（1920x1080）に最適化** でOK
- モバイル対応は不要
